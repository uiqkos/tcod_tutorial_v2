# Project Documentation

This is a simple documentation file for the project.

## Overview

This project is a roguelike game implemented in Python.

## Files

- `main.py`: Entry point of the game.
- `engine.py`: Game engine logic.
- `components/`: Contains game component classes.
- `entity.py`: Entity definitions.
- `game_map.py`: Map generation and handling.

## How to Run

Run the game using:
```
python main.py
```
