# Project Documentation

## Basic Project Structure

- actions.py: Contains game action logic.
- color.py: Defines color constants and utilities.
- components/: Directory for game components.
- data/: Directory for game data files.
- engine.py: Core game engine logic.
- entity.py: Entity definitions.
- entity_factories.py: Factories for creating entities.
- equipment_types.py: Definitions for equipment types.
- exceptions.py: Custom exceptions.
- game_map.py: Game map logic.
- input_handlers.py: Input handling logic.
- main.py: Main entry point of the game.
- message_log.py: Message logging system.
- procgen.py: Procedural generation logic.
- render_functions.py: Rendering functions.
- render_order.py: Defines rendering order.
- setup_game.py: Game setup routines.
- tile_types.py: Tile type definitions.

This documentation provides a high-level overview of the main files and directories in the project.
